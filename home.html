Overview

This project creates a furniture shopping app with live filtering and sorting functionality, and a shopping cart for adding/removing items, as well as changing the quantity of items in the cart and displaying the total price. Such interface is widely used in online shopping websites nowadays, which could largely improve users’ shopping experience with a responsive and interactive design.

The interface created in this project uses React.js and React Bootstrap framework, and practices organizing components and making dynamic changes of data passing flow from scratch.

Usability

Screenshot of the Interface

  • Each item displays in the simple, neat form of a card with item name, filtering/sorting criterias(category, color and price), aggregating category(price) and an add-to-cart button which would not confuse users with overlapped information among items.
  • The default interface displays all the items without any filtering or sorting. When the user hovers over or clicks on the button on the Navbar, it turns dark from light grey. This makes users informed of the filtering criteria they select when seeing the displayed items, and would not get confused with multiple filters.
  • The shopping cart session appears in the right column of the interface, which would keep the user informed of the items selected. If the user wants to add more of an item in the cart, he could just click on the “+” button below the item in the cart section. The “-” button works for decreasing the quantity of remaining items. If uses just want to remove the item, he could also click the remove button beneath. The items will be removed from the cart and the total price gets updated.
